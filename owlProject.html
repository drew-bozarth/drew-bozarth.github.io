Overwatch League Database Application

* This project is a work in progess, expected finish date: December 15th, 2022 *

  • Envisioned an intuitive application for new players that uses Overwatch League statistics to determine the best heroes to play based on different input conditions such as maps, game modes, and teammate hero selections
  • Designed a relational database using MySQL to import all Overwatch League data on players, teams, and matches from 2018-2022
  • Developed an application framework in python to interact with the database and perform CRUD operations in addition to advanced analysis operations to support players
